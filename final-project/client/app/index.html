Welcome to PUP Computer Engineering Thesis Library!

This Theses Library aims to help the PUP Computer Engineering Students in their research for thesis project.

SEARCH

Search previous theses and its desciption for reference on current research. Archived theses can be used as reference of future researchers. They could also search theses that they want to innovate.

ADD

Add your thesis project to the library for archive and for reference. Title, subtitle and description. Edit and Update it if changes occur.

DELETE

Delete old theses that can't be used as a valid reference of future researchers. Disapproved thesis added to this library can also be deleted.
